# 沪深300指数收益率预测系统 — 部署文档

## 1. 项目概述

本系统基于深度学习方法（LSTM、Transformer、LSTM-Transformer）对沪深300指数的对数收益率进行预测，并提供交互式可视化界面。

- **模型训练脚本**: `lz335.py`
- **可视化界面**: `app.py`（Streamlit）
- **数据文件**: `指数行情_000300.xlsx`
- **训练结果**: `results/` 目录

## 2. 环境要求

| 项目 | 最低要求 | 推荐配置 |
|------|---------|---------|
| 操作系统 | Windows 10+ / Linux | Windows 11 / Ubuntu 22.04 |
| Python | 3.8+ | 3.9 ~ 3.11 |
| 内存 | 4 GB | 8 GB+ |
| GPU | 无（CPU可运行） | NVIDIA GPU + CUDA（加速训练） |
| 磁盘 | 500 MB | 1 GB |

## 3. 依赖安装


## 5. 部署步骤

### 5.1 方式一：一键启动（Windows）

双击 `start.bat`，脚本将自动：

1. 检查 Python 环境
2. 检查并安装缺失依赖
3. 检查是否已训练（若 `results/results.json` 不存在则自动运行训练）
4. 启动 Streamlit 服务

启动后浏览器自动打开 `http://localhost:8501`。

### 5.2 方式二：手动启动

**步骤 1：训练模型（首次部署必须执行）**

```bash
python lz335.py
```

训练过程约需 3-10 分钟（视硬件而定），完成后生成 `results/` 目录下的所有文件。

> 若已有 `results/` 目录且无需重新训练，可跳过此步。

**步骤 2：启动可视化界面**

```bash
streamlit run app.py --server.port 8501
```

浏览器访问 `http://localhost:8501` 即可使用。

## 6. 重新训练

如需重新训练模型（例如更换数据或调整参数）：

```bash
# 1. 删除旧结果
rm -rf results/          # Linux/macOS
rmdir /s /q results\     # Windows

# 2. 重新训练
python lz335.py
```

## 7. 关键参数说明

在 `lz335.py` 中可调整的核心参数：

| 参数 | 默认值 | 说明 |
|------|-------|------|
| `LOOKBACK` | 20 | 滑动窗口回看天数 |
| `TARGET_HORIZON` | 1 | 预测未来第几天的收益率 |
| `epochs` | 100 | 训练轮数 |
| `learning_rate` | 0.001 | 学习率 |
| `batch_size` | 64 | 批大小 |
| 训练/验证/测试集比例 | 70% / 15% / 15% | 按时间顺序划分 |
| 滚动验证窗口数 | 3 | 稳健性分析窗口数量 |

模型超参数（在各模型类的 `__init__` 中）：

| 模型 | 参数 | 默认值 |
|------|------|-------|
| LSTM | `hidden_dim=32, num_layers=2, dropout=0.2` | — |
| Transformer | `d_model=64, nhead=4, num_layers=2, dim_feedforward=128, dropout=0.2` | — |
| LSTM-Transformer | `lstm_hidden=64, d_model=32, nhead=2, num_layers=2, dim_feedforward=64, dropout=0.2` | — |

## 8. 数据格式要求

数据文件 `指数行情_000300.xlsx` 需包含以下列：

| 列名 | 类型 | 说明 |
|------|------|------|
| 交易日期 | 日期 | 格式 YYYY-MM-DD |
| 开盘指数 | 浮点数 | 开盘价 |
| 最高指数 | 浮点数 | 最高价 |
| 最低指数 | 浮点数 | 最低价 |
| 收盘指数 | 浮点数 | 收盘价 |
| 成交数量(股) | 整数 | 成交量 |
| 成交金额(元) | 浮点数 | 成交额 |

更换数据时，确保列名一致，或将 `lz335.py` 中 `col_map` 映射调整为实际列名。

## 9. 常见问题

### Q1: 启动后提示"未找到训练结果"

确保已运行 `python lz335.py` 且 `results/results.json` 存在。

### Q2: GPU 相关

系统会自动检测 CUDA 可用性。有 GPU 时训练更快，无 GPU 时使用 CPU 训练（较慢但可正常运行）。

### Q3: 端口被占用

修改启动端口：

```bash
streamlit run app.py --server.port 8502
```

### Q4: 中文乱码

`start.bat` 已设置 `chcp 65001`。若终端仍有乱码，将终端编码设为 UTF-8。

### Q5: 依赖安装失败

尝试逐个安装排查：

```bash
pip install numpy pandas
pip install torch
pip install streamlit plotly
pip install scikit-learn scipy pykalman openpyxl
```

## 10. 系统功能说明

可视化界面包含 6 个标签页：

| 标签页 | 功能 |
|--------|------|
| 数据概览 | 展示 KPI 指标、收盘价走势、收益率分布、描述性统计 |
| 训练过程 | 各模型训练/验证损失曲线对比 |
| 预测对比 | 实际值 vs 预测值、散点图、残差分布、三模型叠加对比 |
| 指标对比 | MSE/RMSE/MAE/MAPE/R2/DA 指标排名、雷达图、柱状图 |
| 未来预测 | 未来 10 个交易日收益率预测、累积收益率、三模型对比 |
| 滚动验证 | 滚动时间窗口稳健性分析、各窗口 R2/DA 对比 |
